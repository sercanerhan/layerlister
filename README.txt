Türkçe:
Katman Listeleyici

ArcMap 'de aktif olan mxd nizde bulunan katmanların "isimlerini" D:\liste.txt 'ye yazdırır.

Yüklemek için:
layerlister.esriaddin dosyasini kullanabilirsiniz.

English:
Layer Lister

Creates a list of active ArcMap's mxd's layers to D:\liste.txt

Installation:
Use layerlister.esriaddin
